You have two convenient shell scripts for running the Groovy scripts included in this sample without bothering with the classpath:
* runGroovyJDK6.sh
* runClosureJDK6.sh 

The sample has been tested using Groovy 2.1.0; please refer to [1] for directions on how to install Groovy.

[1] http://groovy.codehaus.org/Installing+Groovy
